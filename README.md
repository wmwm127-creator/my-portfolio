# 私のポートフォリオサイト

デザイナーとしての制作実績をまとめたポートフォリオサイトです。

## 🌐 公開URL
https://wmwm127-creator.github.io/my-portfolio/

## 🛠 使用技術
- HTML5 / CSS3
- JavaScript
- Adobe Dreamweaver / VS Code (開発エディタ)
- Squoosh (画像最適化)

## ✨ こだわったポイント
- **レスポンシブデザイン**: スマートフォンでもPCでも、作品の魅力が伝わるよう余白とレイアウトを調整しました。
- **パフォーマンス**: 画像をWebP形式に変換・圧縮し、表示速度を意識した実装を行いました。
- **アクセシビリティ**: 適切なタグ選定を行い、情報の構造が正しく伝わるように意識しました。

## 📂 構成について
実務を想定し、CSSや画像などをフォルダごとに整理して管理しています。
